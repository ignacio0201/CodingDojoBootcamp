JavaScript Basics

JavaScript is a cross-platform, object-oriented scripting language. JavaScript is extremely popular for a variety ofreasons. It is a small and lightweight language allowing maximun flexibility for developers to take it in a bunch ofdifferent directions. JavaScript lives inside a hostenviroment (a web beowser or Node server) it can be connected to the objects of these enviroments to provide programmatic control over them.

  • Variable Declaration JavaScriptvariables are containers for storing data values -imagine a cup you fill with coffe, the cup holds thecoffee, a variable holds a value. All JavaScriptvariables must be identified with unique names. Theseunique names are called identifiers. var x;

  • Variable Assignment Assignmentoperators assign values to JavaScript variables - ourcup can now have cpffe poured in it, giving our variablea value to hold. The = assignment operator assigns avalue to a variable. var x = 10;

  • Data Types Data types are an importantconcept; to be able to operate on variables you need toknow the data type. There are six data types that areJavaScript primitives: Boolean - true or false; null - null aka nothing; Number - 42 or 3.14159; String - "Coding Dojo Rocks!"; Array - [1, 'Coding', 2, 'Dojo']; and Object - {first_name: 'Jane', last_name: 'Doe'}

  • Conditionals When you write code, youwant to perform different actions for differentdecisions - hitting different code blocks based onvalues or conditions tha have been met. You can useconditional statements in your code to accomplish this.There are the following conditional statements: if aspecified condition is true, do this code in our codeblock; else if to specify a new condition to test, ifthe fist condition is false; else we execute this blockof code;

  • Loops There are many different kind ofloops in every programming language, but they allessentially do the same thing: they will repeat anaction some number of times. Imagine you have to run amile, well you run around the track four times and thenyou stop. Thats a loop!
